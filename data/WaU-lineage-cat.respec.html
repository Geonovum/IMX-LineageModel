Lineage - detail

The lineage information model consists of three separate parts:

  • A metadata binding model to model the data elements that have lineage information;
  • The lineage model describing the mapping activities by which the orchestrated data elements are derived;
  • The source model describing the data elements that are referenced in the source model.
